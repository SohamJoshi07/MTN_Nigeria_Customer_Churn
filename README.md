📊 Customer Churn Analysis & Prediction
📌 Project Overview
This project focuses on analyzing customer churn data and building a machine learning pipeline to understand the key factors that lead customers to leave a service.
The goal is to extract actionable insights and prepare a foundation for churn prediction models that can help businesses improve customer retention.

🎯 Objectives
Perform Exploratory Data Analysis (EDA) on customer churn data
Identify important features influencing churn
Handle missing values and class imbalance
Build a machine learning-ready pipeline
Prepare the project for real-world deployment and scalability

🧠 Skills & Concepts Used

Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Handling Imbalanced Data (SMOTE)
Machine Learning Pipelines
Model Evaluation
Python & Jupyter Notebook
